---\nname: supabase-api\ndescription: Работа с Supabase базой данных в LinguFlow\n---\n\n# Supabase API\n\n## Таблицы\n- profiles: id (uuid), username, avatar_url\n- decks: id, user_id, title, description, created_at\n- cards: id, deck_id, front, back, created_at\n\n## Политики RLS\n- decks: пользователь видит только свои колоды\n- cards: пользователь видит карточки только из своих колод\n\n## Основные функции (src/lib/db.js)\n- fetchDecks(userId) - получить все колоды пользователя\n- createDeck(userId, title, description) - создать колоду\n- deleteDeck(deckId) - удалить колоду\n- addCard(deckId, front, back) - добавить карточку\n- deleteCard(cardId) - удалить карточку\n- createInitialDecks(userId) - создать начальные колоды (A1-A2, B1-B2, C1-C2)\n\n## Примеры запросов\n`javascript\nconst { data } = await supabase.from('decks').select('*').eq('user_id', userId)\nconst { data } = await supabase.from('cards').select('*').eq('deck_id', deckId)\n`\n\n## Auth\n- Supabase Auth для регистрации/входа\n- user_id связывает данные с пользователем\n
